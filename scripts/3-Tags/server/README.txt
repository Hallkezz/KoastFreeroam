Для выдачи привилегий, добавьте в данную директорию нужные вам .txt файлы и вносите steamid игроков построчно.

Пример:
STEAM_0:0:90087002
STEAM_0:0:172102301
STEAM_0:1:229056007

- admins.txt - список админов.
- adminsD.txt - список донат-админов.
- creators.txt - список владельцев.
- gladmins.txt - список главных админов.
- modersD.txt - список донат-модераторов.
- vips.txt - список випов.
- youtubers.txt - список ютуберов.

specialplayers.lua - настраиваемые префиксы.
